:active

La pseudo-clase (en-US) :active de CSS representa un elemento (como un botón) que el usuario está activando. Cuando se usa un mouse, la "activación" generalmente comienza cuando el usuario presiona el botón primario del mouse y termina cuando se suelta. La pseudo-clase :active se usa comúnmente en los elementos "a" y "button", pero también se puede usar en otros elementos.

Ejemplo:

A los elementos .text y .shape input se les especificaron los siguientes estilos en css:

.text:active{
color: aqua;}

.shape:active{
background-color: aqua;}

Al aplicar la pseudoclase :active, los elementos cambiarán su color a azul (aqua) cuando se haga clic sobre ellos.

:checked

La pseudo-clase :checked de CSS representa cualquier radio (input type="radio" (en-US)), checkbox (input type="checkbox") u option (option en un elemento select) que está marcado o conmutado a un estado on.

Ejemplo:

A la etiqueta input se le especificaron los siguientes estilos en css:

input:checked{
height: 30px;
width: 30px;}

Al aplicar la pseudoclase :checked, se modificará el tamaño de las input cuando estén seleccionadas; o dicho de otra manera, cuando estén en "on".

:default

La pseudo-clase :default de CSS representa cualquier elemento de formulario que sea el predeterminado entre un grupo de elementos relacionados.
Este selector se puede usar en los elementos button, input type="checkbox", input type="radio" y option.

Ejemplo:

A la etiqueta input se le especificaron los siguientes estilos en css:

input:default{
box-shadow: 0 0 10px aqua;}

input:default + label{
color: aqua;}

Al aplicar la pseudoclase :default, se agregará una sombra al input, que se definió como opción por defecto. Su respectivo label aparecerá también de color azul (aqua).

:dir()

La pseudo-clase :dir de CSS coincide con los elementos en función de la direccionalidad del texto contenido en ellos.

ltr:
Orientar elementos de izquierda a derecha.

rtl:
Orientar elementos de derecha a izquierda.

Ejemplo:

A los elementos con la clase .text, que se encuentran dentro de los div se le especificaron los siguientes estilos en css:

div .text:dir(ltr){
color aqua;}

Al aplicar la pseudoclase :dir(ltr), se buscarán los textos que estén direccionados de izquierda a derecha. Los que cumplan con esta condición (ltr) se pintarán de azúl (aqua). Nota: Pseudoclase experimental. Sólo funciona en Mozilla firefox

:disabled

La pseudo-clase :disabled de CSS representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado (por ejemplo ser selecionado, hacer click en él o aceptar texto de entrada) ni aceptar el foco.

Ejemplo

En el siguiente formulario hay un grupo de 3 input. Al elemento input dispuesto para la Dirección, se le especificó en html el atributo disabled, por lo cual, no permitirá la entrada de texto. En css se le dieron estilos a los input que estén en estado disabled:

input :disabled{
background-color: aqua;}

Debido a que es el único input en estado disabled, es el único que se rellena de color azul (aqua).

:empty

La pseudo-clase :empty de CSS representa cualquier elemento que no tenga hijos. Los hijos pueden ser nodos de elemento o texto (incluido el espacio en blanco). Los comentarios o las instrucciones de procesamiento no afectan si un elemento se considera vacío.

Ejemplo

A todos los elementos div de abajo, se les especificaron los siguientes estilos en css:

div{
backgrund-color: green;}

div:empty{
backgrund-color: aqua;}

La pseudoclase :empty hace referencia a los elementos que están vacíos. Como el div de la izquierda no tiene contenido, los estilos se aplican a este y se rellena de color azul (aqua).

:enabled

La pseudo-clase :enabled CSS representa cualquier elemento habilitado. Un elemento está habilitado si puede ser activado (es decir seleccionado, se puede hacer click en él, acepta que se le introduzca texto, etc.) o si accepta el foco. El elemento también tiene un estado deshabilitado el en cuál, no puede ser activado ni acepta el foco.

Ejemplo

En el siguiente formulario hay un grupo de 3 input. Al elemento input dispuesto para la Dirección, se le especificó en html el atributo disabled, por lo cual, no permitirá la entrada de texto. En css se le dieron estilos a los input que estén en estado enabled:

input :enabled{
background-color: aqua;}

Debido a que los input Nombre y Correo se encuentran en estado enabled, se rellenan de color azul (aqua).

:first

La pseudo-clase :first de CSS, utilizada con la regla-at @page, representa la primera página de un documento impreso.

Nota: No puede cambiar todas las propiedades de CSS con esta pseudoclase. Solo puede cambiar los márgenes, orphans (en-US), widows y saltos de página del documento. Además, solo puede usar unidades de longitud absoluta al definir los márgenes. Todas las otras propiedades serán ignoradas.

Ejemplo

Con la pseudoclase :first, se hace referencia a la primera página a imprimir.

@page:first{
margin-top: 150px; }

section{
page-break-after:avoid; }

Al dar clic en el botón inprimir de abajo, se abrirá una ventana para configurar la impresión de este sitio web. La primera página aparecerá con un margin-top (margen superior) de 150px.

:first-child

La pseudo-clase :first-child de CSS representa el primer elemento entre un grupo de elementos hermanos.

Ejemplo

Se establecieron estilos en css para los primeros hijos de los dos contenedores:

.caja1 p:first-child{
color: violet;}

.caja2 h3:first-child{
color: green;}

Usando :first-child Se pintan los primeros hijos de las cajas uno y dos, respectivamente.

:hover

La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor (puntero del mouse).

:invalid

La pseudo-clase :invalid de CSS representa cualquier elemento input u otro elemento form cuyos contenidos no se puedan validar.

Ejemplo:

A los elementos input se les especificaron los siguientes estilos en css:

background-color: pink;}

De esta manera, cuando se ingrese un dato inválido en el campo, este se pintará de color rosa (pink).

:lang()

La pseudo-clase :lang() de CSS es utilizada para modificar elementos en función del idioma en el que se determina que están.

Ejemplo:

A la etiqueta p se le aplicaron los siguientes estilos en css:

p:lang(en){
color:aqua;}

Esto quiere decir que aparecerá en color azul (aqua) el texto que esté establecido en idioma inglés (en).
